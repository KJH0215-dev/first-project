# 📘 C 언어 기초 정리 노트 (개강 전 예습)

## 🧠 1. 컴파일러 vs 인터프리터

### ▪ 컴파일러 (Compiler)

* 소스코드 전체를 한 번에 번역해서 실행 파일을 만든다.
* C 언어는 컴파일 언어이다.
* 예: `gcc`

과정:

```
소스코드(.c) → 컴파일 → 실행파일(.exe) → 실행
```

### ▪ 인터프리터 (Interpreter)

* 코드를 한 줄씩 읽으면서 바로 실행한다.
* 예: Python

---

## ⚙ 2. gcc란?

* GNU Compiler Collection
* C 코드를 컴파일하는 프로그램

사용 예시:

```
gcc firststep.c -o firststep
```

* `firststep.c` : 소스코드 파일
* `-o firststep` : 실행파일 이름 지정

실행:

```
.\firststep.exe
```

---

## 🏗 3. C 프로그램 실행 구조

기본 구조:

```c
#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```

### ▪ #include <stdio.h>

* 표준 입출력 라이브러리 포함
* printf, scanf 사용 가능하게 함

### ▪ int main()

* 프로그램 시작점
* 운영체제가 여기서부터 실행
* int는 종료 시 정수를 반환

### ▪ return 0;

* 정상 종료 의미

---

## 📦 4. 변수의 개념

```c
int x;
x = 10;
```

* `int x;` → 메모리 공간 확보
* `x = 10;` → 그 공간에 값 저장

안전한 순서:

```
선언 → 값 저장 → 사용
```

---

## 🖨 5. printf 정리

기본 형태:

```c
printf("형식", 값);
```

### ▪ %d

* 정수 출력

### ▪ %f

* 실수 출력

### ▪ %c

* 문자 출력

### ▪ %s

* 문자열 출력

### ▪ \n

* 줄바꿈 (newline)

예:

```c
printf("Sum: %d\n", a+b);
```

---

## 🔤 6. 특수 기호 의미 정리

### ▪ ; (세미콜론)

* 문장의 끝을 의미

### ▪ { }

* 코드 블록 범위
* 함수, if문, 반복문 등에 사용

### ▪ &

* 변수의 주소를 의미 (scanf에서 사용)

## 💡 오늘 배운 핵심 개념

* C는 컴파일 언어이다.
* gcc는 C 코드를 실행 파일로 만들어준다.
* 변수는 메모리 공간이다.
* = 는 연결이 아니라 값 복사이다.
* 값을 넣기 전에 사용하면 위험하다.
* printf는 형식을 먼저 써야 한다.
